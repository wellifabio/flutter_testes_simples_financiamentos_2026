# Calculadora de financiamentos
Aplicativo flutter que faz simulações de financiamentos com juros compostos
- Exemplo para aula de flutter com testes automatizados de integração

## Tecnologias
- Flutter
- VsCode
- AndroidStudio
    - Emulador
- [Testes automatizado](./testes.md)
    - Biblioteca integration_test:

## Como testar localmente
- 1 Clone este repositório
- 2 Abra com VsCode, instale as dependências e execute o aplicativo
```bash
flutter pub get
flutter run
```
- 3 Para executar o teste de integração automatizado excute o comando:
```bash
flutter test integration_test/simulacao.dart
```
